# 2 快捷键
快捷键设置：Sequence Hotkeys
- cmd + click 打开链接的note
- cmd + e section间转换
- cmd + o 笔记间转换
- cmd + shift + f 所有笔记搜索
- cmd + option + < backward
- cmd + option + > forward
- cmd + n 新笔记
